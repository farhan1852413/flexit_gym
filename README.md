## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend dependencies required for static site

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/flexit.git
   cd flexit
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your preferred browser
   open index.html
   ```

3. **Or use a local server** (recommended)
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

4. **Navigate to**
   ```
   http://localhost:8000
   ```
